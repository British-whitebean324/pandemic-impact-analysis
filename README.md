## 🖥️ What This App Does

You can use it to:

- View trends in cases, deaths, and recovery
- Compare regions by population
- See charts made with Python and Matplotlib
- Use a rule-based risk check for quick review
- Work through the process with an interactive command-line menu

The app uses a clear OOP design, so the code is split into small parts that each do one job. That helps the app stay organized and easier to run.

## 📊 Features

### 🗂️ Data Handling
The app can read structured COVID-19 data and organize it for analysis. It uses pandas to work with tabular data, so you can inspect rows, columns, and totals.

### 📈 Charts and Visuals
The app uses Matplotlib to create charts that make trends easier to see. These can help you spot rises, drops, and changes across time or regions.

### 🧠 Rule-Based Risk Prediction
The app includes a simple risk model. It checks population-normalized values, such as cases per person, to help rank risk levels.

### 🧩 OOP Design
The app keeps its logic in classes and objects. That setup helps each part of the program stay focused on one task.

### 💬 Interactive CLI
The command-line menu lets you work through the app one step at a time. This is useful if you want clear prompts instead of a full screen dashboard.

### 🌍 Population-Normalized Metrics
Raw case counts can hide the real picture. This app compares data against population size, so large and small regions can be reviewed more fairly.
